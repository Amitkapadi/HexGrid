Detailed Description

FloatAxial represents a pseudo-position on the hex grid. It does not directly represent the position of a hex, but instead is used as a means to compute a hex position. It is needed in order to transform the world-space point into hex coordinates in HexGrid.PointToCubic() which eventually yields a CubicHexCoord from CubicHexCoord.Round().
